# Security Policy

## Supported Versions

Security updates apply to the main branch.

## Reporting a Vulnerability

To report a vulnerability, email:
tpapenb@iu.edu

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fixes (optional)

## Disclosure Timeline

1. Acknowledge report within 48 hours.
2. Investigate within 7 days.
3. Patch or mitigation within 30 days.
4. Public disclosure after fix is released.
